## near-re-tools

Collection of tools for reverse engineering near access targets.

| Name | Description |
| -------- | -------- |
| [BCMHeapVisualisers](./BCMHeapVisualisers) | Broadcom firmware heap visualisers from Project Zero. |
| [BCMPatcher](./BCMPatcher) | Broadcom firmware patcher from Project Zero. |
| [dump-pci](./dump-pci) | script to dump PCI ring information from Broadcom firmware snapshot from Project Zero. |
| [dump-timers](./dump-timers) | script to dump the timer list from a Broadcom firmware snapshot from Project Zero. |
